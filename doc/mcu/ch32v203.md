# 关于 ch32v203

亮点:

支持 2 个 USB 12Mbps 接口, 以及 4 个 UART. 比 `ch32v103` 更便宜 !

## 芯片介绍

参考资料: <https://www.wch.cn/products/CH32V203.html>

- 指令集: `rv32imac`

- 20KB SRAM, 64KB flash, CPU 主频 144MHz

- 封装: LQFP48, TSSOP20, LQFP32, QFN28, QFN48

- 工作电压: 2.5/3.3V

- 接口:
  - **USB 12Mbps x2** (FS)
  - **UART x4**
  - I2C x2
  - SPI X2
  - GPIO x37
  - CAN x1
  - 12bit ADC x10
  - OPA x2

- 价格 (仅芯片 CNY): 约 2.6 元 (ch32v203c8t6 LQFP48)
