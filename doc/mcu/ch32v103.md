# 关于 ch32v103

亮点:

支持 USB 12Mbps 接口, 以及 3 个 UART.

## 芯片介绍

参考资料: <https://www.wch.cn/products/CH32V103.html>

- 指令集: `rv32imac`

- 20KB SRAM, 64KB flash, CPU 主频 80MHz

- 封装: LQFP48, LQFP64, QFN48

- 工作电压: 2.7 ~ 5.5V

- 接口:
  - **USB 12Mbps** (FS)
  - **UART x3**
  - I2C x2
  - SPI x2
  - GPIO x51
  - 12bit ADC x16

- 价格 (仅芯片 CNY): 约 3.1 元 (ch32v103c8t6 LQFP48)
