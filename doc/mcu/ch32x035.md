## 芯片介绍

参考资料: <https://www.wch.cn/products/CH32X035.html>

- 指令集: `rv32imac`

- 20KB SRAM, 62KB flash, CPU 主频 48MHz

- 封装: TSSOP20, LQFP48, LQFP64M, QFN20, QFN28, QSOP28

- 工作电压: 3.3V/5V

- 接口:
  - **USB 12Mbps** (FS)
  - **UART x4**
  - I2C x1
  - SPI x1
  - GPIO x46
  - 12bit ADC x10
  - OPA x2
  - CMP x3
  - PIOC x1
